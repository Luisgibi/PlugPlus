PlugPlus
=====

Plug.dj script plugin for Chrome

Current Features
------------------------
* Featured Rooms
* AutoWoot
* AutoJoin


Planned Features
------------------------
* Bot Controlls (TBD)
* Mod Features (TBD)
* Language Filter
* Notifications (Working on)


Possible Features
------------------------
* Custom Themes for any room that requests it. (It will only work for your room)
* Skin Overriding (only you can see them)
