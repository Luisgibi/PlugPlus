Global Notifications Uncheck this to disable all notifications completely. (Currently only works on Windows, Mac, and ChromeOS)
Chat Level
Choose what chats you would like to be notified of. Disable all options to disable. Using the all option will override the others.
User Level
Choose what degree of notification you would like for users leaving, joining, and voting.
Song Update
Choose what songs you would like to be notified of.
DJ Update
Select what DJs you would like to be notified of.
AutoWoot Delay Choose a number in seconds to wait before wooting. (0-90)
